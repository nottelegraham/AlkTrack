DIY alkalinity trend tester description
Purpose: describe what the tester does, what it does not do, what the formula means, and what hardware is required.
Description
This device is a trend-based alkalinity monitor, not a laboratory-grade alkalinity tester and not a true acid titrator. It estimates tank alkalinity movement by comparing an aerated aquarium sample against an aerated reference sample whose starting alkalinity was manually entered from a hobby titration test.
The reference sample and tank sample are sequentially exposed to the same air source for the same aeration period, then measured with the same pH probe and temperature probe. The formula subtracts the estimated borate alkalinity from the known reference alkalinity, scales the remaining carbonate alkalinity by the carbonate-speciation ratio implied by the two stabilized pH readings, and then adds back the estimated borate alkalinity for the tank sample.
Because both samples are treated under the same conditions, room-air CO2 is treated as a shared variable that mostly cancels for trend-monitoring purposes. This does not mean all CO2-related variables are perfectly eliminated; it means the method is designed to be repeatable enough to show whether alkalinity is rising, falling, or staying stable over time.
The tester should be viewed as a repeatable alkalinity trend estimator. It is not intended to replace occasional manual alkalinity checks, it is not expected to produce perfectly absolute dKH values, and its output should not be used as the sole basis for aggressive automated dosing decisions without independent confirmation.
